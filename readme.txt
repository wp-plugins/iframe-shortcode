== Description ==

Allows you to paste iframe codes in the visual editor, just enclose within the [iframe][/iframe] shortcode.

== Installation ==

1. Use automatic installer
1. Activate plugin
1. When editing a post or a page, to add an iframe - type [iframe] then paste code, then type [/iframe] to enclose.
1. When the post is viewed, the iframe code will be converted to display properly.

== Frequently Asked Questions ==

= Can I use YouTube, Vimeo, SlideShare, SlideRocket, etc... iframe embed codes? =
Honey badger don't care. Any iframe inside the shortcode will be decoded and displayed. Go wild.
